# Remote Version Hosting

This folder mirrors the public release metadata hosted by GitHub Releases.

- `latest.json` is uploaded as a release asset and downloaded by `Launcher.exe`.
- `Overlay.zip` is generated by the release pipeline from `releases/package`.
- `changelog.txt` is uploaded for humans and copied into `latest.json`.

The launcher requires `Overlay.zip` to contain `Overlay.exe` at the archive root.
